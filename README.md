# Jammming App with React

## Project Description

*I created this project as a part of the **Codecademy** Full-stack developer path to practice. This project is a **playlist web application with the Spotify API** which can be accessed through any web browser to search for songs and to create and save playlists to one's Spotify account.*

---

### 1. Key features

- The **single-page-app** is built with **React**. HTML and CSS was provided. The app uses an **API** from Spotify to create and save playlists.

### 2. Usage

- You can **access** the application [here](http://unknown-art.surge.sh/) in any browser with an internet connection; no need to install anything.
- You can enter an artist, song title or an album to the *search bar* and then click on the *SEARCH* button.
  > ⚠️ When you use the app **for the first time**, you will be asked to **login** to your Spotify account (only once). After successful login, your search results will be shown in the *Results* area.
- In the *Results* section then you can click ton the *'+'* button to add songs to your *New Playlist*.
You can rename your new playlist by clicking on the title.
- You can remove songs from the playlist with the *'-'* button.
- When you are happy with your new playlist, you can save it on your Spotify account by clicking on the *SAVE TO SPOTIFY* account button. Then you will see your new playlist in your Spotify account under your playlists.
